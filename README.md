# 茅山奇门排盘系统 (MQC)

## 1. 技术底座
- **内嵌天文数据**：系统内嵌了 1600-3500 年的高精度 TDB (历书时) 数据。
- **历算法则**：严格遵循真太阳时校正，内置二十四节气（精准至秒）与朔望月数据引擎。

## 2. 核心算法逻辑
- **排盘体系**：支持茅山排盘法，兼容“转盘”与“飞盘”双重布局切换。
- **干支审计**：实时计算年、月、日、时四柱干支，并根据节气交节时刻动态确定局目。
- **物理场映射**：通过内嵌的 `solarTermMap` 与 `moonPhaseMap`，将天文运行轨迹精准映射至奇门九宫。

## 3. 使用说明
- **环境要求**：纯 HTML5/Javascript 开发，无需服务器，任何现代浏览器均可运行。
- **操作流程**：选择自定义时间 -> 选择排盘方式（转盘/飞盘） -> 系统自动检索万年历数据并渲染全息图像。
